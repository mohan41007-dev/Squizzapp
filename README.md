Squizzapp: AI-Driven Quiz & Lead Generation Tool

This is a custom Python application built with customtkinter designed to engage users through interactive assessments. It is part of a professional AI consulting tech stack used to automate outreach and qualify leads for AI-driven email campaigns.

Key Features:

Modern UI: Built using a sleek, dark-themed interface.

Dynamic Logic: Separates application logic from the UI for easy scaling.

Consultant-Ready: Designed to be integrated into LinkedIn outreach and SaaS business workflows.
